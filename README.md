目标玩家：文明，群星等4x玩家，钢四，帝国时代4等即时战略, 模拟, 策略玩家。
背景设定：蓝本是《三体——黑暗森林》，模拟不同宇宙文明之间的扩张，战斗和消亡，以及对宇宙的影响。
玩法独特性：科技树下的多种宇宙规律武器的模拟。

核心目标：点满科技
故事：略
技术：unity2d
美感：统一ui,像素风格模拟，极简。
机制：有策略的点击方格，使用各种探测手段与攻击防御手段赢得游戏。

计划如何提供游戏体验：通过玩家与ai之间的战斗获取乐趣，尝试使用不同的武器和手段赢得游戏。
